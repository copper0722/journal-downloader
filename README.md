# Journal Downloader

Chrome extension for journal table-of-contents parsing, metadata export, and open-access PDF download.

## Public-Use Boundary

- Download only PDFs that are explicitly open access, public-domain, government-published, or otherwise clearly licensed for public download.
- Treat subscription, institutional, login-only, or unknown-access items as metadata-only.
- Do not read browser cookie stores, password managers, tokens, or authenticated sessions.
- Do not use this project to bypass access controls.

## Output

The extension can export article metadata including title, DOI, article type, links, visible abstract, OA status, and source URL. Public repos should store metadata and public-safe summaries, not copyrighted full text or subscription PDFs.
